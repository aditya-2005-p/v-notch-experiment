Introduction

The coefficient of discharge (Cd) is a dimensionless number that represents the ratio of the actual discharge to the theoretical discharge through a notch.

Formulas

For a V-notch: Q = (8/15) * Cd * H^(5/2)

For a Rectangular notch: Q = (2/3) * Cd * L * H^(3/2)
